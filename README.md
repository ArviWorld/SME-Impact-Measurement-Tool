# SME Impact Measurement Tool
The SME Impact Measurement Tool is a downloadable Excel-based framework designed to help start-ups, SMEs, and impact-driven projects measure and communicate their greenhouse gas (GHG) emissions and mitigation performance. It provides transparent, comparable, and decision-useful insights for investors, management, and stakeholders, supporting sustainability-driven decision-making from early to growth stages.

# Download the Tool
The tool is available as a macro-enabled Excel file:

**- Download:** *SME_Impact_Measurement_Tool.xlsm* from this repository.
**Please enable macros after downloading to ensure full functionality.**

## Scope & Purpose
- Measure GHG emissions (CO₂e) across lifecycle stages
- Compare baseline vs. project scenarios
- Track emission reductions and mitigation performance
- Support ESG reporting and stakeholder communication

## Methodology & Standards
The tool is fully or partially aligned with:
- GHG Protocol
- ISO 14064
- ISO 14040/44 (Life Cycle Assessment)
- IPCC Guidelines
- Recognized regional and international datasets

Calculations are based on:
- Primary activity data (user input)
- Verified emission factors

## Input Requirements
Provide data for both baseline and project scenarios:
- Raw materials
- Manufacturing (energy, water)
- Transportation (sourcing, distribution)
- End-of-life treatment

## Limitations
- Only fossil-based CO₂e included
- Biogenic CO₂ not included
- Not a substitute for third-party verified GHG accounting
- Results depend on data quality and system boundary
